# GitHub iOS Test

هذا مشروع بسيط حتى تتعلم GitHub Actions.

## ماذا يفعل
عند تشغيل الـ workflow سيقوم GitHub بـ:
1. تشغيل macOS
2. إنشاء ملف تجريبي
3. ضغطه باسم TestApp.ipa
4. إعطائك الملف للتحميل

## كيف تشغله
1. ارفع الملفات إلى GitHub
2. افتح تبويب Actions
3. اختر Build Test IPA
4. اضغط Run workflow
5. بعد انتهاء البناء ستجد ملف IPA في Artifacts
